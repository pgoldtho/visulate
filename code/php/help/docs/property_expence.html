Property Expenses

This screen is for managers and owners. All the expenses for a given property are recorded here

Start where it says Details and click on new expense. Write a description and click save, then click on “new receipt or invoice” to record who did the work and what they charged, and the expense type. Click on save again. You can records all the invoices in one place that go with the original description. For example when remodeling a kitchen there could be a number of different invoices making up the total job. The list at the top of the page details the different expenses for a given property, click on the circle to the left to see all the invoices associated with that expense which will appear at the bottom of the screen.
